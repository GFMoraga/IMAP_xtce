# IMAP: CoDICE XTCE Builder
This is a repo that consists of building xml/xtce formatted files with python

## Getting Started
### Prerequisites
* Python 3.9
* ElementTree
* Pandas

### Installation
1. Clone the repo
   ```sh
   git clone
    ```
2. Install Python packages
3. Run the script
   ```sh
   python3.9 main.py
   ```
## Usage
### Input
* Input file: `input.xlsx`
* Input file sheet: `input`

### Output
* Output file: `output.xml`
